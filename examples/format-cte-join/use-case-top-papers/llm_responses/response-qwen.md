## ⚠️ Limitations & Caveats

1. **Publication lag**: Peer-reviewed journals often have 6-18 month delays; arXiv preprints are more current
2. **Author name variations**: Many winners publish under different name formats (e.g., "G. Parisi" vs "Giorgio Parisi")
3. **Collaborative papers**: Recent work often has many co-authors; identifying "lead" contributions requires domain expertise
4. **Scope**: This answer provides representative samples; a truly exhaustive list would require querying 30+ individual academic profiles

For the most accurate, personalized publication lists, I recommend:
- Visiting each researcher's institutional homepage (e.g., Peter Scholze: mpim-bonn.mpg.de/scholze) [[105]]
